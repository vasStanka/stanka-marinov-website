Pose2Drone – Human‑Drone Interaction Pose2Drone – Mensch‑Drohnen‑Interaktion

A skeleton‑based Human‑Drone Interaction framework that lets users control a drone’s motion and maintain safe distance using simple arm gestures, complemented by a monocular distance‑estimation technique that avoids depth sensors. Accepted at IEEE EUSIPCO 2021. Ein skeletonbasiertes Framework für Mensch-Drohnen-Interaktion, das Benutzern erlaubt, die Bewegung einer Drohne mit einfachen Armbewegungen zu steuern und einen sicheren Abstand zu halten, ergänzt durch eine monokulare Entfernungsschätzung ohne Tiefensensoren. Veröffentlicht auf der IEEE EUSIPCO 2021.

Paper Code

Lightweight Renderer Effizienter Renderer

A lightweight, CPU‑C++ ray tracer. It supports OBJ loading, BVH acceleration, and advanced path‑tracing features such as Multiple Importance Sampling, Next Event Estimation, and Practical Path Guiding. It runs on Linux and offers a real‑time ImGui interface for camera viewpoint, materials, lightning, and other rendering settings. Ein effizienter CPU C++-Raytracer. Er unterstützt OBJ-Import, BVH-Beschleunigung und erweiterte Path Tracing Funktionen wie Multiple Importance Sampling, Next Event Estimation und Practical Path Guiding. Läuft unter Linux und bietet eine Echtzeit ImGui Oberfläche für Kameraperspektive, Materialien, Beleuchtung und andere Rendering-Einstellungen.

  • Tools: imgui, glfw, glm, tinyexr
